SmartMonitor custom UI project

Files:
- *.ui: encrypted vendor-style UI source
- config.ini: startup image settings
- images/background.png: initial background

Workflow:
1. Edit the UI source or replace images.
2. In configure.py choose SmartMonitor HID.
3. Click 'Convert UI->DAT' and select this .ui file.
4. Save and run to upload the compiled theme.
